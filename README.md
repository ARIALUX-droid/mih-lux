# mih-lux

一个自用的 Mihomo (Clash.Meta) 裸核运行小方案。

本仓库的内容主要是在 豆包 的协助下完成的，主打一个“懒人专属”。主要是为了解决在 Android (已 Root) 和 Windows 环境下，手动折腾裸核太琐碎的问题。

做了点什么：
Android 端：针对 KernelSU 环境写了几个简单的 Shell 脚本，实现了内核自动下载、更新和静默启动，不用每次手动去 GitHub 捞包。

Windows 端：整了一套对应的启动/停止脚本，保持操作习惯和手机端一致。

### 运行逻辑
* **极简部署**：只需要在脚本所在的同一文件夹下，放置好 **Mihomo 内核文件** 和 **配置文件 (config.yaml)**，运行脚本即可实现代理。
* **简易覆写**：启动脚本内支持添加订阅链接，启动时会自动写入配置文件，方便随时更换。（不需要可自行删除或注释）
* **自动 TUN**：安卓端必须开启 TUN 模式才能正常代理，脚本在启动时会自动检测并开启该功能。
  
一点私货：顺便存了份自己用的配置和规则集。

代码写得比较随性，主要是为了自己用着顺手。如果你也喜欢这种“裸核”运行的简洁感，希望能给你一点参考。
